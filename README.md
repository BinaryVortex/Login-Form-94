# Login Form

A clean, responsive login form built with HTML, CSS and JavaScript.


Preview

![Login Form Preview](https://raw.githubusercontent.com/BinaryVortex/Login-Form-94/main/Screenshot%202024-08-06%20085020.png)


Description

This repository contains a simple, accessible login form intended as a frontend UI example and learning project. It demonstrates layout, form styling, basic validation, and user-focused design using only HTML, CSS and a bit of JavaScript.

Features

- Responsive layout that works on desktop and mobile
- Styled form with focus and hover states
- Basic client-side validation with JavaScript
- Easy to customize and integrate into larger projects

Files

- index.html — the login form markup
- style.css — styles for the form and layout
- Screenshot 2024-08-06 085020.png — preview image used in this README

Getting started

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-94.git

2. Open index.html in your browser (double-click or serve with a local web server).

How to use

- Enter a username/email and password, then click Sign In.
- The form includes basic client-side checks — it is not wired to a real backend.

Customize

- Modify style.css to change colors, spacing, or typography.
- Update index.html to add additional inputs or change the structure.
- Replace the JavaScript checks with your own validation or connect the form to an API.
